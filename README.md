# 🍽️ LocalBite

LocalBite is a food delivery web application that allows users to order food from nearby restaurants within a limited delivery radius.

## 🚀 Features

- 📍 Select nearby service areas
- 🏪 Browse restaurants
- 🍔 View menu items
- 🛒 Add to cart
- 💳 Cash on Delivery
- 📦 Order tracking
- 🔐 Admin dashboard

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB Atlas
